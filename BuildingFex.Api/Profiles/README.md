# Profiles — Perfil extendido del administrador

**Responsable:** _asignar_

## Nota

En BuildingFex, muchos datos del admin ya están en `User` (dni, company, ruc, address).
Este módulo es **opcional**: úsalo si quieres separar credenciales (IAM) de datos de perfil (Profiles), como en learning-center-platform.

Si no lo necesitan, pueden dejar todo en IAM y marcar este módulo como "no usado".
